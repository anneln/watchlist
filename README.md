# Movie Watchlist

This is a solo project to create a movie watchlist.

## What should the app do ?

The user searches for a film title via the search bar.
If the film is in the database, it is displayed below the search bar.
Each film card has a button to add it to the watchlist.
When the user clicks that button, another page opens to display the watchlist.
The user can remove a film from the watchlist.

## Technical Requirements

- [x] Follow figma design
- [x] Connect to OMDb database via its API
- [x] Save data to local storage
